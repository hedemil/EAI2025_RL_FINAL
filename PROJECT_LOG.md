# Formatting rules for logs:
* **Dates**
    * yy/mm/dd
* **File**
    * Relative path
* **Function/Code Block**
    * Part of code, if relevant
*   **Description**
    * Short description of what has been done
# Example:
# Date - File - Function/Code Block
- Description

# 25/09/18 - PROJECT_LOG.md - None
- Initialized Project Log
